# SkyMate
An end-to-end AI-powered flight tracking platform using Next.js, FastAPI, MySQL, SQLAlchemy, OpenAI API, and MCP-based tool orchestration, enabling real-time flight monitoring, weather-aware travel insights, trip management, and intelligent assistant-driven actions.


Backend:
  cd backend
  source venv/bin/activate
  uvicorn app.main:app --reload

Frontend:
  cd frontend
  run dev
